FuelFit — Guía de configuración del proveedor de IA

Cómo obtener una clave API y habilitar el análisis de fotos de alimentos con IA

100% Transparencia: Nunca queremos que se lleve una sorpresa. Así es como FuelFit maneja sus datos:
  • Sin servidor backend — todos sus datos permanecen encriptados en su dispositivo únicamente
  • Llamadas API directas — las llamadas de IA van desde su dispositivo directamente al proveedor que elija
  • Consentimiento explícito — usted elige activar la IA durante el onboarding; puede omitirlo por completo
  • Un interruptor para minimizar la exposición de datos — el Coach de dieta con IA se puede desactivar en Configuración. Con él desactivado, su proveedor solo es contactado cuando usted toma activamente una foto de comida; sus números nutricionales diarios nunca se envían de forma automática para el coaching. El análisis de fotos en sí se ejecuta solo cuando elige tomar una foto, por lo que está iniciado por el usuario por diseño.
  • Etiqueta de IA honesta — cuando el coaching recurre a una plantilla integrada (interruptor apagado, sin clave o fallo de la IA), la insignia "IA" desaparece para que siempre sepa qué es realmente generado por el modelo
  • Exportar/Importar — sus datos son portátiles a través de Configuración → Exportar/Importar

¿Cuánto cuesta?

FuelFit utiliza IA para analizar sus fotos de comida. Usted trae su propia clave API (BYOK); FuelFit no cobra suscripciones. El costo depende completamente de qué proveedor elija y con qué frecuencia registre sus comidas.

Descargo de responsabilidad y seguridad de facturación:
  • Control de precios: Todas las tarifas de API, recuento de tokens y estimaciones de costos se proporcionan solo como referencia. Los proveedores de IA (OpenAI, Alibaba Cloud, Mistral) controlan sus propios precios y pueden cambiar sus tarifas en cualquier momento. FuelFit no cobra suscripciones, no procesa la facturación y no tiene control sobre los cargos de las API.
  • Límites de facturación: Recomendamos encarecidamente establecer límites de facturación estrictos (límites de uso) en el panel de desarrollador de su proveedor de IA para evitar cargos inesperados en caso de un uso excesivo de la aplicación o filtración de claves.
  • Descargo de responsabilidad médica: FuelFit y su coach de IA son solo para fines informativos y educativos. No constituyen asesoramiento médico, diagnóstico dietético ni tratamiento profesional. Consulte siempre con un profesional de la salud calificado antes de realizar cambios significativos en su dieta o rutina de ejercicio.

Opción 1: Qwen / Alibaba Cloud (Predeterminado — Más barato)

Qwen (DashScope) — qwen3-vl-flash

Más barato · Predeterminado

Qwen es el proveedor predeterminado de FuelFit. El modelo qwen3-vl-flash es el modelo de visión dedicado más barato disponible y maneja las fotos de comida especialmente bien, incluyendo la cocina china. Las nuevas cuentas de Alibaba Cloud obtienen créditos gratuitos.

Configuración paso a paso

1
Crear una cuenta de Alibaba Cloud
Vaya a account.alibabacloud.com/register (internacional) o account.aliyun.com (China) y regístrese.
2
Activar Model Studio (百炼)
Vaya a bailian.console.aliyun.com y active el servicio Model Studio. Las nuevas cuentas obtienen créditos gratuitos (1 millón de tokens), suficiente para meses de uso normal.
3
Crear una clave API
En la consola de Model Studio → API Key Management → haga clic en "Create API Key" → copie la clave (empieza con sk-).
4
Introducir en FuelFit
Abra FuelFit → Configuración → Servicios de IA → asegúrese de que Qwen esté seleccionado (es el predeterminado). La URL base y el modelo vienen precompletados; cambie qwen3-vl-flash a qwen3-vl-plus si desea una mayor precisión. Pegue su clave y pulse Verificar conexión.
~$0.69 / year
Costo estimado para usuario activo (6 fotos + 12 llamadas al coach/día)
Nivel gratuito: Las nuevas cuentas reciben 1 millón de tokens gratuitos, suficiente para meses de uso. El modelo qwen3-vl-flash está diseñado específicamente para la comprensión de imágenes y es el modelo de visión más barato disponible.

Opción 2: OpenAI (Más confiable)

OpenAI — gpt-4o-mini

Más confiable

OpenAI ofrece una sólida confiabilidad y un amplio soporte en el ecosistema. Es un poco más caro que Qwen, pero su comportamiento es extremadamente predecible; un valor predeterminado sólido si no se encuentra en China.

Configuración paso a paso

1
Crear una cuenta de OpenAI
Vaya a platform.openai.com/signup y regístrese con su correo electrónico o cuenta de Google/Microsoft.
2
Agregar un método de pago
Navegue a Configuración → Facturación y agregue una tarjeta de crédito. Puede comenzar con tan solo $5.
3
Crear una clave API
Vaya a API Keys → haga clic en "Create new secret key" → asígnele un nombre (ej. "FuelFit") → haga clic en "Create key".
4
Copiar y guardar la clave
Copie la clave de inmediato (empieza con sk-). No podrá volver a verla. Guárdela en un lugar seguro.
5
Introducir en FuelFit
Abra FuelFit → Configuración → Servicios de IA → seleccione OpenAI. La URL base https://api.openai.com/v1 y el modelo gpt-4o-mini están precompletados. Pegue su clave y pulse Verificar conexión.
~$1.45 / year
Costo estimado para usuario activo (6 fotos + 12 llamadas al coach/día)

Option 4: Personalizado (Cualquier endpoint compatible con OpenAI)

Custom Provider

Usuario avanzado

Si desea utilizar un proveedor que no está integrado — Anthropic Claude, Google Gemini, un servidor vLLM propio, OpenRouter como pasarela de múltiples proveedores o cualquier otra API compatible con OpenAI — elija Custom e introduzca el endpoint usted mismo.

Por qué Anthropic no está integrado: El adaptador oficial compatible con OpenAI de Anthropic no ha sido confiable para las solicitudes de visión (las imágenes pueden perderse silenciosamente). Si desea utilizar Claude para fotos de comida, dirija las solicitudes a través de OpenRouter como proveedor personalizado; es la vía más confiable hoy en día.

Configuración paso a paso

1
Obtener una clave API del proveedor elegido
Por ejemplo: OpenRouter, Google AI Studio o su propio servidor.
2
Buscar la URL base compatible con OpenAI
La mayoría de los proveedores documentan esto. Ejemplos: OpenRouter es https://openrouter.ai/api/v1; el endpoint compatible con OpenAI de Google es https://generativelanguage.googleapis.com/v1beta/openai.
3
Elegir un modelo que admita visión
FuelFit necesita un modelo multimodal que acepte imágenes. Ejemplos en OpenRouter: anthropic/claude-3.5-sonnet, google/gemini-2.0-flash, openai/gpt-4o-mini.
4
Introducir en FuelFit
Abra FuelFit → Configuración → Servicios de IA → seleccione Custom → complete Nombre, Base URL, Model y API Key → pulse Verificar conexión.
El costo varía enormemente según el proveedor y el modelo. OpenRouter publica una tabla de precios por token para cada modelo de su plataforma.

Interruptor del Coach de dieta con IA

Las dos funciones de IA de FuelFit tienen perfiles de flujo de datos muy diferentes, y solo una de ellas necesita un interruptor:

  • El análisis de fotos lo inicia el usuario. No se envía nada al proveedor hasta que pulsa el botón de la cámara. Si no toma una foto, no se realiza ninguna llamada API. No hay nada que desactivar.
  • El Coach de dieta con IA se ejecuta automáticamente. Cuando abre la pestaña de Insights o se actualiza la tarjeta del coach en la pantalla de inicio, FuelFit envía su resumen diario de nutrición al proveedor para generar un mensaje de coaching. Ese es exactamente el tipo de llamada en segundo plano que algunos usuarios no desean por motivos de privacidad (incluso a un proveedor elegido por ellos mismos). Por eso esta función cuenta con un interruptor.

Apagar el Coach de dieta con IA mantiene el análisis de fotos en perfecto funcionamiento, pero detiene cualquier otra llamada API automática en segundo plano. La tarjeta del coach en Insights cambiará de forma silenciosa a consejos basados en plantillas locales que se calculan completamente en su dispositivo.

Transparencia de datos

  • Qué datos se envían a dónde:
  • Análisis de fotos de alimentos: se envían la foto y un breve prompt directamente al proveedor de IA que eligió. Nada más.
  • Coach de dieta con IA (interruptor ENCENDIDO): solo se envían los números de su resumen de nutrición diario (calorías, proteínas, carbohidratos, grasas, fibra, azúcar, sodio, agua, recuento de comidas). Sin fotos, sin detalles del perfil, sin datos históricos más allá de hoy.
  • Coaching basado en plantillas (interruptor APAGADO o fallo de IA): los mensajes se calculan por completo en el dispositivo a partir de sus datos registrados. Nada sale de su teléfono.

La insignia "IA" — Atribución honesta

En la pestaña Insights, la tarjeta del coach muestra una pequeña etiqueta "IA" solo cuando el mensaje provino realmente de su proveedor de IA. Cuando el mensaje se crea a partir de plantillas locales (porque el interruptor está apagado, no tiene clave API o la llamada de IA falló), la insignia desaparece. De este modo, siempre sabe si lo que lee ha sido generado por un modelo o por la lógica propia de FuelFit.

Cómo desactivar el coaching de IA (Configuración)

1
Abrir Configuración
Pulse la pestaña Configuración en FuelFit.
2
Desplegar "Servicios de IA"
Pulse el acordeón de Servicios de IA. Verá la configuración de su proveedor y, debajo, el interruptor del Coach de dieta con IA.
3
Apagar el interruptor
Con el interruptor desactivado, verá consejos de coaching basados en plantillas en lugar de los generados por IA, y la insignia "IA" desaparecerá de la tarjeta del coach en Insights. El análisis de fotos sigue funcionando de forma normal.

Durante el Onboarding

Durante la pantalla de configuración de IA en el onboarding, tiene dos caminos:

  • Configurar un proveedor y verificar — verá el interruptor del Coach de dieta con IA en la pantalla siguiente, configurado de manera predeterminada en ENCENDIDO. Puede desactivarlo allí si solo desea el análisis de fotos.
  • Omitir por ahora — la pantalla del interruptor del Coach de dieta con IA se omite por completo (al no haber IA configurada, no hay nada que controlar). El interruptor se configura en APAGADO automáticamente y un banner le recuerda que las funciones de IA están desactivadas. Puede configurar un proveedor y activar el interruptor más adelante en Configuración.
Nota de privacidad: con el interruptor del Coach de dieta con IA apagado, su proveedor solo es contactado cuando toma activamente una foto de comida. No se envían datos para el coaching ni para los resúmenes diarios; estos se producen mediante plantillas integradas que se ejecutan por completo en su dispositivo.

Preguntas frecuentes (FAQ)

¿Es segura mi clave API?

Sí. Su clave API se almacena de forma segura en el almacenamiento seguro nativo de su dispositivo (iOS Keychain / Android Keystore a través de capacitor-secure-storage-plugin) y nunca se guarda en el almacenamiento local estándar ni en IndexedDB. FuelFit no tiene servidor backend; es una aplicación puramente del lado del cliente. Todas las llamadas de IA van directamente desde su dispositivo al proveedor que eligió. Nunca vemos ni recopilamos sus datos.

¿Qué pasa al cambiar de dispositivo?

FuelFit incluye una función integrada de Exportar/Importar en Configuración. Exporte sus datos a un archivo y luego impórtelo en su nuevo dispositivo. Sus comidas, configuraciones y claves API (encriptadas) se transferirán en un solo paso.

¿Puedo usar la IA solo para fotos y no para el coaching?

Sí. Abra Configuración → Servicios de IA, despliegue la sección y desactive el interruptor del Coach de dieta con IA. Esto desactivará los mensajes de coaching generados por IA manteniendo activo el análisis de fotos con IA.

¿Puedo usar varios proveedores?

FuelFit utiliza un proveedor principal a la vez, pero puede configurar varios proveedores personalizados y alternar entre ellos en Configuración. Los tres proveedores integrados (Qwen, OpenAI, Mistral) también recuerdan su propia configuración, por lo que cambiar entre ellos es solo un toque.

¿Qué pasa si me quedo sin créditos?

La aplicación mostrará un error al intentar analizar una foto o cuando el Coach de dieta con IA intente generar un mensaje. Si la llamada al coach falla, la tarjeta de Insights recurrirá silenciosamente a un consejo de plantilla (sin alertas invasivas, y la insignia "IA" desaparecerá). Añada créditos a la cuenta de su proveedor y la IA volverá a funcionar. Su historial de comidas siempre se guarda localmente.

¿Necesito una clave API para usar FuelFit?

No. FuelFit funciona sin IA para el registro manual de comidas. Solo necesita una clave API si desea el análisis de fotos o el coaching generado por IA. Sin clave configurada, todo el coaching utilizará las plantillas integradas.

¿Qué proveedor debo elegir?

  • Predeterminado: Qwen qwen3-vl-flash — el más barato, gran reconocimiento de comida, créditos gratis para nuevas cuentas
  • Más confiable: OpenAI gpt-4o-mini — comportamiento predecible, amplio soporte de ecosistema
  • Basado en la UE: Mistral mistral-small-latest
  • Cualquier otro (Claude, Gemini, local): use la opción Custom, preferiblemente a través de OpenRouter

¿Cómo cambio de proveedor?

Vaya a Configuración → Servicios de IA → pulse sobre un proveedor diferente en la lista → introduzca (o pegue) la clave API para ese proveedor → pulse Verificar conexión.

¿Por qué se puede editar el campo de modelo para proveedores integrados?

Los valores predeterminados están optimizados para el costo. Si desea mayor precisión, puede preferir qwen3-vl-plus en Qwen, gpt-4o en OpenAI, o un modelo diferente de Mistral. Los campos Base URL y Model son precompletados pero editables.
