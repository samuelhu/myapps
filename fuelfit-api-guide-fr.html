FuelFit — Guide de configuration du fournisseur d'IA

Comment obtenir une clé API et activer l'analyse photo des aliments par l'IA

100 % Transparence : Nous ne voulons jamais que vous soyez pris au dépourvu. Voici comment FuelFit gère vos données :
  • Pas de serveur backend — toutes vos données restent chiffrées sur votre appareil uniquement
  • Appels API directs — les requêtes IA vont directement de votre appareil au fournisseur choisi
  • Consentement explicite — vous choisissez d'activer l'IA lors de l'onboarding ; vous pouvez ignorer cette étape
  • Un interrupteur pour minimiser l'exposition des données — le Coach alimentaire IA peut être désactivé dans les Paramètres. Lorsqu'il est désactivé, votre fournisseur n'est contacté que lorsque vous prenez activement une photo de repas — vos données nutritionnelles quotidiennes ne sont jamais envoyées sans action de votre part. L'analyse photo s'exécute uniquement lorsque vous choisissez de prendre une photo, elle est donc initiée par l'utilisateur par conception.
  • Badge IA honnête — lorsque le coaching repasse sur un modèle local (interrupteur éteint, pas de clé ou échec IA), le badge "IA" disparaît pour que vous sachiez toujours ce qui a été réellement généré
  • Exporter/Importer — vos données sont portables via Paramètres → Exporter/Importer

Combien ça coûte ?

FuelFit utilise l'IA pour analyser vos photos de nourriture. Vous apportez votre propre clé API (BYOK) — il n'y a pas de frais d'abonnement de la part de FuelFit. Le coût dépend entièrement du fournisseur que vous choisissez et de la fréquence de vos enregistrements.

Pourquoi environ 12 appels coach par jour ? L'application met en cache les réponses du coach localement. Elle n'appelle pas l'API si vos données enregistrées restent identiques. Cependant, ajouter un repas, mettre à jour l'eau ou le changement d'heure invalident immédiatement le cache. Un utilisateur actif enregistrant 6 photos, 4 verres d'eau et ouvrant l'application à 2 heures différentes effectuera environ 12 appels.

  • Avertissement médical : FuelFit et son coach IA sont fournis uniquement à des fins informatives et éducatives. Ils ne constituent pas un avis médical, un diagnostic diététique ou un traitement professionnel. Consultez toujours un professionnel de la santé qualifié avant de modifier votre alimentation ou votre routine sportive.

Option 1: Qwen / Alibaba Cloud (Par défaut — le moins cher)

Qwen (DashScope) — qwen3-vl-flash

Le moins cher · Par défaut

Qwen est le fournisseur par défaut de FuelFit. Le modèle qwen3-vl-flash est le modèle de vision dédié le moins cher disponible et reconnaît particulièrement bien la nourriture, y compris la cuisine asiatique. Les nouveaux comptes Alibaba Cloud bénéficient de crédits gratuits.

Configuration étape par étape

1
Créer un compte Alibaba Cloud
Allez sur account.alibabacloud.com/register (international) ou account.aliyun.com (Chine) et inscrivez-vous.
2
Activer Model Studio (百炼)
Rendez-vous sur bailian.console.aliyun.com et activez le service Model Studio. Les nouveaux comptes reçoivent des crédits gratuits (1 million de tokens) — de quoi tenir plusieurs mois d'utilisation normale.
3
Créer une clé API
Dans la console Model Studio → API Key Management → cliquez sur "Create API Key" → copiez la clé (commence par sk-).
4
Saisir dans FuelFit
Ouvrez FuelFit → Paramètres → Services IA → assurez-vous que Qwen est sélectionné. L'URL de base et le modèle sont pré-remplis — augmentez vers qwen3-vl-plus pour plus de précision. Collez votre clé et appuyez sur Vérifier la connexion.
~$0.69 / year

Option 4: Personnalisé (Tout point de terminaison compatible OpenAI)

Custom Provider

Utilisateur avancé

Si vous souhaitez utiliser un fournisseur non listé par défaut (Claude d'Anthropic, Gemini de Google, un serveur vLLM local, OpenRouter comme passerelle, ou toute autre API compatible OpenAI), choisissez Custom.

Pourquoi Anthropic n'est pas intégré par défaut : L'adaptateur compatible OpenAI officiel d'Anthropic s'est révélé instable pour les requêtes avec images (les photos peuvent être ignorées sans erreur). Si vous souhaitez utiliser Claude pour l'analyse de repas, passez par OpenRouter configuré comme fournisseur personnalisé.

Configuration étape par étape

1
Obtenir une clé API de votre fournisseur
Par exemple : OpenRouter, Google AI Studio, ou votre serveur local.
2
Trouver l'URL de base compatible OpenAI
Exemples : OpenRouter est https://openrouter.ai/api/v1 ; l'URL compatible de Google est https://generativelanguage.googleapis.com/v1beta/openai.
3
Sélectionner un modèle avec vision
FuelFit nécessite un modèle multimodal acceptant les images. Exemples sur OpenRouter : anthropic/claude-3.5-sonnet, google/gemini-2.0-flash, openai/gpt-4o-mini.
4
Saisir dans FuelFit
Ouvrez FuelFit → Paramètres → Services IA → sélectionnez Custom → remplissez Name, Base URL, Model, et API Key → appuyez sur Vérifier la connexion.
Le coût varie fortement selon le fournisseur et le modèle. OpenRouter publie la grille tarifaire des tokens pour chaque modèle disponible sur sa plateforme.

Interrupteur du Coach alimentaire IA

Les deux fonctionnalités d'IA de FuelFit ont des flux de données différents, et une seule nécessite un interrupteur :

  • L'analyse photo est déclenchée par l'utilisateur. Rien n'est envoyé à votre fournisseur tant que vous n'appuyez pas sur le bouton de la caméra. Si vous ne prenez pas de photo, aucun appel API n'a lieu. Il n'y a rien à couper.
  • Le Coach alimentaire IA fonctionne en arrière-plan. Lorsque vous ouvrez l'onglet Insights ou que la carte du coach s'actualise sur l'accueil, FuelFit transmet un résumé nutritionnel quotidien pour générer les conseils. Pour des raisons de confidentialité, certains utilisateurs préfèrent éviter ces appels automatiques en arrière-plan. Cette fonction dispose donc d'un interrupteur.

Désactiver le Coach alimentaire IA maintient l'analyse photo fonctionnelle, mais coupe tout autre appel API automatique en arrière-plan. La carte du coach dans Insights repasse alors sur des conseils locaux pré-enregistrés, générés entièrement par votre téléphone.

Fonctionnement des fonctionnalités

Fonctionnalité Ce qu'elle fait Quand elle appelle votre fournisseur Comment couper les appels
Analyse photo des aliments Identifie les composants et estime les valeurs nutritionnelles depuis les photos prises Uniquement quand vous appuyez sur le bouton de l'appareil photo Ne prenez pas de photos (ou effacez votre clé API dans les Paramètres pour désactiver l'intégration).

Transparence des données

  • Quelles données sont partagées :
  • Analyse photo : La photo et une consigne courte sont envoyées directement au fournisseur d'IA choisi. Rien d'autre.
  • Coach alimentaire IA (interrupteur ACTIVÉ) : Seuls vos totaux nutritionnels du jour sont partagés — calories, protéines, glucides, lipides, fibres, sucre, sodium, eau, nombre de repas. Pas de photos, pas d'infos de profil, pas d'historique.
  • Coaching local (interrupteur DÉSACTIVÉ ou échec de l'IA) : Les conseils sont calculés sur votre téléphone à partir de vos données locales. Rien ne quitte votre appareil.

Le badge "IA" — Transparence de l'auteur

Dans l'onglet Insights, la carte affiche un petit badge "IA" uniquement si le conseil provient de votre fournisseur d'IA externe. Si la recommandation est basée sur un modèle local (parce que l'interrupteur est désactivé, que vous n'avez pas configuré de clé ou que l'appel a échoué), le badge disparaît. Vous savez ainsi toujours ce que vous lisez.

Comment désactiver le coaching par IA (Paramètres)

1
Ouvrir les Paramètres
Appuyez sur l'onglet Paramètres dans FuelFit.
2
Déplier "Services IA"
Appuyez sur l'accordéon Services IA. Vous verrez votre configuration et, juste en dessous, l'interrupteur du Coach alimentaire IA.
3
Désactiver l'interrupteur
Une fois désactivé, vous verrez s'afficher les conseils locaux pré-enregistrés, et le badge "IA" disparaîtra. L'analyse photo continue de fonctionner normalement.

Pendant l'onboarding

Durant l'écran de configuration de l'IA lors de l'onboarding, vous avez deux options :

  • Configurer un fournisseur et vérifier — l'interrupteur du Coach alimentaire IA apparaîtra sur l'écran suivant (activé par défaut). Vous pouvez le désactiver immédiatement si vous ne souhaitez que l'analyse de photos.
  • Ignorer pour l'instant — l'écran de l'interrupteur est sauté (sans clé configurée, le coaching IA ne peut fonctionner). L'interrupteur est réglé sur DÉSACTIVÉ automatiquement et un bandeau vous rappelle que les fonctions d'IA sont éteintes. Vous pouvez configurer tout cela plus tard dans les Paramètres.
Note de confidentialité : lorsque le Coach alimentaire IA est désactivé, votre fournisseur d'IA n'est contacté que lorsque vous prenez activement une photo de nourriture. Aucune donnée n'est envoyée pour le coaching ou les résumés quotidiens — ceux-ci sont créés par des modèles locaux s'exécutant entièrement sur votre téléphone.

Foire Aux Questions (FAQ)

Ma clé API est-elle sécurisée ?

Oui. Votre clé API est stockée de manière sécurisée dans le stockage sécurisé natif de votre appareil (iOS Keychain / Android Keystore via capacitor-secure-storage-plugin) et n'est jamais stockée dans le stockage local standard ni dans IndexedDB. FuelFit n'a aucun serveur backend — c'est une application purement côté client. Tous les appels IA partent directement de votre appareil vers le fournisseur choisi. Nous ne voyons ni ne collectons jamais vos données.

Qu'en est-il du changement de téléphone ?

FuelFit propose une option de Transfert (Export/Import) intégrée dans les Paramètres. Exportez vos données dans un fichier, puis importez-le sur le nouvel appareil. Vos repas, réglages et clés API (chiffrées) sont transférés en une seule étape.

Puis-je utiliser l'IA pour les photos uniquement, pas pour le coaching ?

Oui. Allez dans Paramètres → Services IA, dépliez la section, et désactivez l'interrupteur Coach alimentaire IA. Les messages de conseils d'arrière-plan seront locaux, mais l'analyse photo restera active.

Puis-je configurer plusieurs fournisseurs ?

FuelFit utilise un fournisseur principal à la fois, mais vous pouvez configurer plusieurs profils dans Custom et basculer de l'un à l'autre dans les Paramètres. Les trois fournisseurs intégrés (Qwen, OpenAI, Mistral) conservent également leur propre configuration en mémoire.

Que se passe-t-il si je n'ai plus de crédits chez mon fournisseur ?

L'application affichera une erreur lors de l'analyse d'une photo ou lorsque le Coach tentera de générer un message. Si l'appel coach échoue, la carte d'Insights basculera silencieusement vers un conseil local (sans alerte intrusive, et le badge "IA" s'effacera). Rechargez votre compte chez le fournisseur pour que tout fonctionne à nouveau.

Faut-il obligatoirement une clé API pour utiliser FuelFit ?

Non. FuelFit fonctionne parfaitement sans IA pour le suivi manuel de repas. La clé API n'est requise que pour l'analyse photo ou le coaching IA. Sans clé configurée, le coaching utilise les conseils locaux par défaut.

Quel fournisseur dois-je choisir ?

  • Par défaut : Qwen qwen3-vl-flash — le moins cher, excellente vision, crédits gratuits offerts aux nouveaux comptes
  • Le plus fiable : OpenAI gpt-4o-mini — comportement très stable et prévisible, écosystème éprouvé
  • Basé en Europe : Mistral mistral-small-latest
  • Autre (Claude, Gemini, local) : Utilisez l'option Custom, idéalement via OpenRouter

Comment changer de fournisseur ?

Allez dans Paramètres → Services IA → appuyez sur un autre fournisseur dans la liste → saisissez (ou collez) la clé API → appuyez sur Vérifier la connexion. Vos réglages et votre historique sont conservés.

Pourquoi le champ de modèle est-il modifiable pour les fournisseurs intégrés ?

Les modèles par défaut sont configurés pour optimiser les coûts. Si vous préférez une plus grande précision, vous pouvez utiliser qwen3-vl-plus sur Qwen, gpt-4o sur OpenAI ou un autre modèle Mistral. Les champs URL et Modèle sont modifiables.
