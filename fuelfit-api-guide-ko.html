API 키를 획득하고 AI 식사 사진 분석을 활성화하는 방법

100% 투명성: 사용자 몰래 과금되는 현상을 방지하기 위해, FuelFit의 데이터 처리 방침을 명확히 공개합니다:
  • 백엔드 서버 없음 — 모든 데이터는 사용자의 기기 내에만 암호화되어 안전하게 보관됩니다
  • 직접 API 호출 — AI 요청은 기기에서 사용자가 설정한 제공업체로 직접 전송됩니다
  • 명시적 동의제 — 온보딩 과정에서 AI 사용 여부를 직접 결정할 수 있으며, 완전히 건너뛸 수도 있습니다
  • 데이터 공유 최소화 스위치 — 설정에서 AI 식단 코치 기능을 비활성화할 수 있습니다. 비활성화 시 사용자가 직접 카메라로 사진을 찍어 식단을 기록할 때만 서버를 호출하며, 코칭 메시지 생성을 위해 식단 정보가 백그라운드로 자동 전송되지 않습니다. 사진 분석 역시 사용자가 직접 촬영할 때만 실행되므로 완벽히 사용자 주도로만 작동합니다.
  • 정직한 AI 라벨 — 기기 내부의 템플릿 기능으로 대체되어 조언을 제공할 때(스위치 오프, API 키 미설정, AI 오류 등) 카드에서 'AI' 배지가 사라집니다. 사용자는 본인이 읽고 있는 조언이 AI가 생성한 것인지 로컬 템플릿이 제공한 것인지 바로 구분할 수 있습니다.
  • 내보내기/가져오기 — 설정 → 내보내기/가져오기를 통해 데이터를 파일 형태로 안전하게 백업 및 이동할 수 있습니다

이용 요금은 얼마인가요?

FuelFit은 식사 사진 분석에 AI를 활용합니다. 사용자가 자신의 API 키를 직접 등록하여 사용하는 방식(BYOK)으로, FuelFit 애플리케이션 자체 이용료나 구독료는 발생하지 않습니다. 비용은 선택한 AI 제공업체와 사용량에 따라 부과됩니다.

면책 사항 및 결제 안전 조치:
  • 가격 책정: 위의 API 요금율, 토큰 소모량 및 연간 추정 금액은 참고용입니다. AI 제공업체(OpenAI, Alibaba Cloud, Mistral)는 자체 가격 정책을 고수하며 사전 고지 없이 가격을 변경할 수 있습니다. FuelFit은 결제를 대행하지 않으며 수수료를 취하지 않습니다.
  • 이용 한도 설정: API 키의 유출이나 과다한 데이터 전송으로 인한 요금 폭탄을 막기 위해, 각 AI 제공업체의 개발자 대시보드에서 **월 이용 한도(Hard Limit)**를 반드시 설정할 것을 강력히 권장합니다.
  • 의료 면책 조항: FuelFit 앱 및 AI 식단 코치의 피드백은 영양 지식 전달 및 건강 관리 지원 목적으로만 제공됩니다. 의료적 진단, 식사 치료, 의학적 처방을 대신하지 않습니다. 식단 및 운동 계획을 급격히 변경할 때는 의사 등 보건 의료 전문가와 상담하시기 바랍니다.

옵션 1: Qwen / Alibaba Cloud (기본 권장 — 가장 저렴함)

Qwen (DashScope) — qwen3-vl-flash

가장 저렴함 · 기본값

Qwen은 FuelFit의 기본 제공업체입니다. qwen3-vl-flash 모델은 현재 시각 분석 모델 중 요금이 가장 저렴하면서도 한식을 비롯한 동아시아 요리를 아주 훌륭하게 찾아냅니다. Alibaba Cloud 신규 가입 시 무료 크레딧을 받을 수 있습니다.

설정 방법

1
Alibaba Cloud 계정 생성
account.alibabacloud.com/register(글로벌) 혹은 account.aliyun.com(중국 사이트)에 접속해 가입합니다.
2
Model Studio (百炼) 서비스 신청
bailian.console.aliyun.com 에 접속하여 백련(Model Studio) 서비스를 신청해 활성화합니다. 신규 가입자에게 수개월 분량에 달하는 100만 토큰의 무료 혜택이 바로 지급됩니다.
3
API 키 만들기
Model Studio 콘솔 화면의 → API Key Management 메뉴로 이동 → "Create API Key" 단추 클릭 → 생성된 키(sk-로 시작함)를 복사합니다.
4
FuelFit 앱에 등록
FuelFit 앱에서 설정 → AI 서비스 로 이동하여 제공업체에 Qwen을 선택합니다. Base URL 및 Model 필드는 이미 기입되어 있습니다. 더 높은 인식률을 시험하려면 모델명을 qwen3-vl-plus로 수정하여 사용하세요. 복사한 API 키를 붙여넣고 연결 확인을 탭합니다.
~$0.69 / year
액티브 유저 기준 예상 연간 요금 (하루 사진 6회 + 코칭 12회 호출 기준)
무료 혜택: 신규 사용자에게 100만 무료 토큰을 지급하여 오랜 기간 무료로 테스트할 수 있습니다. qwen3-vl-flash는 시각 식별 전용으로 나와 비용 대비 효율성이 매우 높습니다.

옵션 2: OpenAI (가장 안정적임)

OpenAI — gpt-4o-mini

가장 안정적임

OpenAI는 업계에서 가장 신뢰받고 널리 쓰입니다. Qwen에 비해 비용은 소폭 높으나 결과물의 품질과 답변 형식이 매우 정교하고 안정적으로 유지됩니다.

설정 방법

1
OpenAI 계정 생성
platform.openai.com/signup 에서 이메일이나 소셜 아이디로 계정을 만듭니다.
2
결제 수단 및 충전 설정
설정 → 결제(Billing) 메뉴로 이동하여 신용카드를 등록하고 선불 충전(최소 5달러 이상)을 진행합니다.
3
API 키 만들기
API Keys 메뉴로 이동 → "Create new secret key" 클릭 → 이름(예: FuelFit) 입력 → "Create key" 클릭.
4
키 복사 및 보관
키(sk-로 시작함)를 복사하여 안전한 위치에 기록해 둡니다. 보안을 위해 화면을 닫으면 해당 키값은 재차 표시되지 않습니다.
5
FuelFit 앱에 등록
FuelFit 앱에서 설정 → AI 서비스 로 이동하여 제공업체에 OpenAI를 선택합니다. 채워진 URL(https://api.openai.com/v1)과 모델(gpt-4o-mini) 확인 후 키를 입력해 연결 확인을 탭합니다.
~$1.45 / year

옵션 4: Custom (그 외 OpenAI 표준 규격 지원 모델)

Custom Provider

고급 사용자용

내장 제공업체 외에 Anthropic Claude, Google Gemini, 직접 호스팅한 vLLM 모델, 그리고 다중 서비스 게이트웨이인 OpenRouter 등 타사 API를 이용하고 싶은 경우 Custom을 택하여 정보를 입력합니다.

Anthropic Claude가 내장되어 있지 않은 이유: Anthropic사의 자체 OpenAI 호환 어댑터가 이미지 업로드 요청 시 가끔 이미지를 유실시키는 문제가 확인되었습니다. 식사 사진 분석에 Claude 모델을 쓰시려면 OpenRouter를 통해 Custom으로 연동하시는 편이 훨씬 안전하고 신뢰도가 높습니다.

설정 방법

1
사용할 서비스의 API 키 획득
예: OpenRouter, Google AI Studio 또는 개인 서버 등.
2
해당 서비스의 OpenAI 호환 규격 Base URL 획득
대부분의 업체가 안내서를 제공합니다. 예: OpenRouter는 https://openrouter.ai/api/v1, 구글 AI 스튜디오는 https://generativelanguage.googleapis.com/v1beta/openai 입니다.
3
이미지(Vision)를 지원하는 모델명 확인
식단 추정을 위해 이미지를 입력받는 멀티모달 모델이 필요합니다. 예: anthropic/claude-3.5-sonnet, google/gemini-2.0-flash, openai/gpt-4o-mini 등.
4
FuelFit 앱에 등록
FuelFit 앱에서 설정 → AI 서비스 로 이동하여 제공업체에 Custom을 선택합니다. 이름, Base URL, Model, API Key를 각각 기입하고 연결 확인을 탭합니다.
이용 요금은 모델에 따라 크게 상이합니다. 각 모델별 토큰 단가는 OpenRouter 등 해당 서비스 대시보드에서 확인할 수 있습니다.

AI 식단 코치 스위치

FuelFit에 제공되는 두 개의 AI 기능은 서버를 호출하고 데이터를 전송하는 방식이 상이하며, 이에 따라 한 가지 기능에만 On/Off 스위치를 제공합니다:

  • 식사 사진 분석 (사용자 시작형): 카메라로 사진을 찍어 전송할 때만 서버를 호출합니다. 사진을 찍어 올리지 않으면 어떠한 전송도 일어나지 않아 스위치 통제가 필요 없습니다.
  • AI 식단 코치 (자동 백그라운드 호출형): '분석' 탭을 누르거나 홈 화면 코치 카드가 갱신될 때 FuelFit은 오늘의 총 영양 섭취 정보를 기반으로 코칭 의견을 구합니다. 일부 사용자는 보안상 이러한 백그라운드 호출이 자동으로 일어나는 것을 차단하고 싶어 합니다. 따라서 이 기능에 제어 스위치를 부착해 두었습니다.

AI 식단 코치 스위치를 꺼두어도 사진 촬영을 통한 음식 분석은 완벽하게 동작합니다. 코치 의견만 로컬 템플릿(휴대폰 내부 연산으로 동작)에 의한 조언 텍스트로 자연스럽게 교체됩니다.

기능별 데이터 흐름 요약

기능 설명 서버 호출 시점 호출을 막는 방법
식사 사진 분석 찍어 올린 사진 속 음식을 인식하여 영양소 섭취 비율 추정 카메라 버튼을 누르고 촬영하여 전송할 때만 사진 촬영을 하지 마세요. (설정에서 키를 지워 연동을 끊을 수도 있습니다.)
AI 식단 코치 기록된 식단 데이터를 모아서 피드백 및 조언 생성 자동 실행 — 분석 탭 실행, 홈 화면 카드 자동 갱신 및 특정 트리거 시 스위치 끄기 설정 → AI 서비스 → AI 식단 코치 토글 Off. 로컬 내장 템플릿으로 조언이 교체됩니다.

데이터 수집 및 안전성

  • 어떤 데이터가 가나요:
  • 사진 분석: 찍은 사진 파일과 이를 해석하기 위한 짧은 명령어(프롬프트)만 전송되며, 다른 정보는 일체 나가지 않습니다.
  • 식단 코치 (On 상태): 오늘의 영양소 총합 데이터(칼로리, 단백질, 탄수화물, 지방, 식이섬유, 당류, 나트륨, 수분, 식사 횟수)만 전송됩니다. 사진이나 신체 프로필, 오늘 이전의 식단 기록은 전송되지 않습니다.
  • 로컬 템플릿 코치 (Off 혹은 AI 서버 오류 상태): 사용자의 기기 내에서만 연산이 돌아갑니다. 기기 외부로 데이터가 유출되지 않습니다.

“AI” 배지 — 정직한 출처 표기

분석 탭 코치 카드 상단에 표시되는 조언 메시지가 실제 외부 AI에 의해 실시간 계산된 것일 때만 조그마한 "AI" 표시 배지가 활성화됩니다. 스위치를 꺼두거나 키에 잔액이 없어 기기 내부 로컬 템플릿에 의해 구성된 추천 의견으로 채워진 경우에는 "AI" 배지가 자동으로 비활성화되어 조언의 출처를 정직하게 밝힙니다.

AI 식단 코치를 비활성화하는 방법 (설정)

1
설정 열기
FuelFit 하단 메뉴에서 "설정" 탭으로 이동합니다.
2
"AI 서비스" 누르기
AI 서비스 아코디언 메뉴를 탭하면, 제공업체 입력란 아래로 "AI 식단 코치" 토글 버튼이 보입니다.
3
토글 스위치 끄기
스위치를 비활성화하면 AI 연산 대신 기기 내장 템플릿 기반 피드백이 노출되며 카드 옆 "AI" 마크가 소멸합니다. 사진 분석은 기존대로 계속 이용이 가능합니다.

온보딩 과정 중 흐름

앱을 처음 실행하여 AI 설정 단계를 거칠 때 두 가지 경로가 제공됩니다:

  • 제공업체를 구성하고 연결을 확인한 경우 — 다음 단계 화면에서 AI 식단 코치 스위치(기본 On)를 만나게 되며, 원치 않을 경우 거기서 미리 비활성화해둘 수 있습니다.
  • 건너뛰기를 선택한 경우 — 스위치 확인 단계가 자동으로 생략되고 코치 기능은 Off 상태로 기본값 세팅됩니다. AI 기능이 잠겨 있음을 알리는 안내 배너가 표시되며 추후 설정에서 언제든지 활성화가 가능합니다.
개인정보 참고 사항: AI 식단 코치 스위치가 꺼진 상태에서는, 사용자가 직접 카메라 버튼을 탭해 사진을 전송하지 않는 이상 서버를 호출하지 않습니다. 코칭 및 요약 피드백은 100% 사용자의 기기 로컬 템플릿으로 안전하게 계산됩니다.

자주 묻는 질문 (FAQ)

API 키는 안전하게 보호되나요?

네. API 키는 기기의 네이티브 보안 저장소(capacitor-secure-storage-plugin을 통한 iOS Keychain / Android Keystore)에 안전하게 저장되며, 일반 로컬 저장소나 IndexedDB에는 저장되지 않습니다. FuelFit은 백엔드 서버가 존재하지 않는 순수 클라이언트 사이드 앱입니다. 모든 AI 호출은 기기에서 선택하신 제공업체로 직접 전송됩니다. 개발사에서 사용자의 데이터나 키를 수집하거나 확인하는 것은 불가능합니다.

휴대폰을 변경할 때 데이터는 어떻게 되나요?

설정 메뉴 내부에 "내보내기/가져오기" 기능이 있습니다. 데이터를 파일 형식으로 백업하여 새 휴대폰으로 가져오면 기존 식단 기록, 사용자 설정 및 암호화된 API 키까지 그대로 이전됩니다.

코칭 서비스는 꺼두고 식사 사진 분석용으로만 AI를 쓸 수 있나요?

네, 가능합니다. 설정 → AI 서비스 탭의 AI 식단 코치 토글 스위치만 꺼두시면 사진 분석을 위한 Vision API 기능만 활성화된 상태로 원활히 이용하실 수 있습니다.

여러 업체의 API 키를 저장해 두고 교대로 사용 가능한가요?

활성화되는 서비스는 항상 하나뿐이지만, 설정 화면에서 내장된 3개 업체(Qwen, OpenAI, Mistral)의 키가 각각 기억되므로 터치 한번만으로 제공업체를 신속히 교대하여 쓸 수 있습니다.

API 잔액이 소진되면 어떻게 되나요?

사진 촬영 시 에러가 출력되거나 식단 코칭 메시지가 갱신되지 못하고 로컬 템플릿으로 대체되어 카드 옆 "AI" 마크가 소멸합니다. 제공업체 대시보드에 금액을 충전해 주시면 즉시 정상화되며, 로컬 식단 기록은 여전히 안전하게 보존됩니다.

FuelFit을 사용하려면 API 키가 반드시 있어야 하나요?

아닙니다. API 키를 등록하지 않아도 수동 식단 일지 작성, 칼로리 기록 및 목표 관리는 평생 완전 무료로 가능합니다. 사진 분석이나 AI 실시간 조언이 필요할 때만 키를 등록해 사용해 주세요.

어느 제공업체를 택하는 것이 좋을까요?

  • 기본값: Qwen qwen3-vl-flash — 요금이 제일 저렴하며 아시아권 음식 인식률 우수, 신규 100만 무료 토큰 지원
  • 품질 우선: OpenAI gpt-4o-mini — 가장 전형적이며 응답의 안정성이 검증됨
  • 유럽 권역: Mistral mistral-small-latest
  • 기타 (Claude, Gemini 등): Custom을 활용해 OpenRouter를 통해 가입 및 연결하시길 추천합니다

제공업체를 다른 곳으로 변경하려면 어떻게 하나요?

설정 → AI 서비스 화면으로 가셔서 목록에서 변경하고 싶은 제공업체를 누르고, 새 API 키를 등록하고 연결 확인을 눌러주면 완료됩니다.

왜 내장 제공업체의 모델 필드가 수정 가능하게 열려 있나요?

기본 내장된 모델명은 요율 대비 성능이 우수한 모델로 저희가 추천해 둔 것입니다. 더 우수한 화질 해석을 원하신다면 Qwen의 qwen3-vl-plus 등으로 상향 수정하거나 OpenAI의 gpt-4o를 입력하여 테스트할 수 있도록 편집 가능하게 개방해 두었습니다.
