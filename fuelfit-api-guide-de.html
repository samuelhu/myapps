FuelFit — KI-Dienstleister Einrichtungsanleitung

Wie Sie einen API-Schlüssel erhalten und die KI-Lebensmittelfotoanalyse aktivieren

100% Transparenz: Wir möchten nicht, dass Sie überrascht werden. So verarbeitet FuelFit Ihre Daten:
  • Kein Backend-Server — alle Ihre Daten verbleiben ausschließlich verschlüsselt auf Ihrem Gerät
  • Direkte API-Aufrufe — KI-Anfragen gehen direkt von Ihrem Gerät an den von Ihnen gewählten Anbieter
  • Explizites Opt-in — Sie entscheiden sich während des Onboardings für die Aktivierung der KI; Sie können dies auch komplett überspringen
  • Ein Schalter zur Minimierung der Datenfreigabe — der KI-Ernährungscoach kann in den Einstellungen deaktiviert werden. Wenn er ausgeschaltet ist, wird Ihr Anbieter nur kontaktiert, wenn Sie aktiv ein Lebensmittelfoto aufnehmen – Ihre täglichen Ernährungsdaten werden niemals ungefragt für das Coaching gesendet. Die Fotoanalyse selbst läuft nur, wenn Sie sich für ein Foto entscheiden, und ist somit vom Benutzer initiiert.
  • Ehrliches KI-Label — wenn das Coaching auf ein integriertes Template zurückgreift (Schalter aus, kein Schlüssel oder KI-Fehler), verschwindet das "KI"-Badge, sodass Sie immer wissen, was tatsächlich generiert wurde
  • Export/Import — Ihre Daten sind über Einstellungen → Export/Import portabel

Wie viel kostet es?

FuelFit nutzt KI, um Ihre Lebensmittelfotos zu analysieren. Sie bringen Ihren eigenen API-Schlüssel mit (BYOK) — es gibt keine Abonnementgebühr von FuelFit selbst. Die Kosten hängen vollständig davon ab, welchen Anbieter Sie wählen und wie oft Sie Mahlzeiten protokollieren.

KI-Ernährungscoach-Schalter

Die beiden KI-Funktionen in FuelFit haben sehr unterschiedliche Datenflussprofile, und nur eine davon benötigt einen Schalter:

  • Fotoanalyse ist benutzerinitiiert. Es wird nichts an Ihren Anbieter gesendet, bis Sie auf die Kamera-Schaltfläche tippen. Wenn Sie kein Foto machen, erfolgt kein API-Aufruf. Es gibt nichts zu deaktivieren.
  • Der KI-Ernährungscoach läuft im Hintergrund. Wenn Sie den Insights-Tab öffnen oder die Coach-Karte auf dem Startbildschirm aktualisiert wird, sendet die App Ihre tägliche Nährstoffzusammenfassung an Ihren Anbieter, um eine Nachricht zu generieren. Manche Benutzer möchten diese Hintergrundaufrufe aus Datenschutzgründen verhindern. Deshalb hat diese Funktion einen Schalter.

Das Ausschalten des KI-Ernährungscoaches lässt die Fotoanalyse voll funktionsfähig, stoppt jedoch alle automatischen Hintergrund-API-Aufrufe. Die Coach-Karte wechselt stattdessen lautlos zu integrierten Tipps auf Basis von Vorlagen, die vollständig auf Ihrem Gerät berechnet werden.

Was jede Funktion tut

Feature Was es tut Wann der Anbieter kontaktiert wird Wie die Aufrufe gestoppt werden
Lebensmittelfoto-Analyse Identifiziert Lebensmittel und schätzt Nährwerte anhand von Fotos, die Sie aufnehmen Nur wenn Sie aktiv auf die Kamera-Schaltfläche tippen Nehmen Sie keine Fotos auf (oder löschen Sie Ihren API-Schlüssel).
KI-Ernährungscoach Generiert Coaching-Nachrichten und tägliche Zusammenfassungen aus Ihren Daten Automatisch — beim Öffnen von Insights, beim Aktualisieren der Home-Coach-Karte und bei anderen Auslösern Schalter aus Einstellungen → KI-Dienste → KI-Ernährungscoach. Die Karte wechselt zu lokalen Vorlagen-Tipps.

Daten-Transparenz

  • Welche Daten wohin gesendet werden:
  • Fotoanalyse: Das Foto sowie eine kurze Texteingabe werden direkt an den von Ihnen gewählten KI-Anbieter gesendet. Sonst nichts.
  • KI-Ernährungscoach (Schalter AN): Nur Ihre täglichen Nährwertsummen werden gesendet — Kalorien, Eiweiß, Kohlenhydrate, Fett, Ballaststoffe, Zucker, Natrium, Wasser, Anzahl der Mahlzeiten. Keine Fotos, keine Profil-Details, keine historischen Daten über den heutigen Tag hinaus.
  • Vorlagen-Coaching (Schalter AUS oder KI-Fehler): Die Nachrichten werden vollständig auf dem Gerät aus Ihren Daten berechnet. Nichts verlässt Ihr Telefon.

Das "KI"-Badge — Ehrliche Kennzeichnung

Auf dem Insights-Tab zeigt die Coach-Karte nur dann ein kleines "KI"-Label an, wenn die Nachricht tatsächlich von Ihrem KI-Anbieter stammte. Wenn die Nachricht aus lokalen Vorlagen erstellt wurde – weil der Schalter aus ist, Sie keinen API-Schlüssel haben oder der KI-Aufruf fehlgeschlagen ist –, verschwindet das Badge. So wissen Sie immer, ob das, was Sie lesen, von einem Modell oder von der Logik von FuelFit generiert wurde.

Wie Sie den KI-Coach deaktivieren (Einstellungen)

1
Einstellungen öffnen
Tippen Sie in FuelFit auf den Tab Einstellungen.
2
Erweitern Sie "KI-Dienste"
Tippen Sie auf das Akkordeon. Sie sehen Ihre Anbieterkonfiguration und darunter den Schalter für den KI-Ernährungscoach.
3
Schalter ausschalten
Bei ausgeschaltetem Schalter sehen Sie lokale, vorlagenbasierte Coaching-Tipps statt KI-generierter, und das "KI"-Badge verschwindet von der Insights-Coach-Karte. Die Fotoanalyse funktioniert weiterhin normal.

Während des Onboardings

Während des AI-Setup-Bildschirms beim Onboarding haben Sie zwei Möglichkeiten:

  • Einen Anbieter konfigurieren und überprüfen — Sie sehen dann auf dem nächsten Bildschirm den KI-Ernährungscoach-Schalter, der standardmäßig auf EIN steht. Sie können ihn dort ausschalten, wenn Sie nur die Fotoanalyse wünschen.
  • Vorläufig überspringen — der Bildschirm für den KI-Ernährungscoach-Schalter wird komplett übersprungen. Der Schalter wird automatisch auf AUS gesetzt und ein Banner erinnert Sie daran, dass die KI-Funktionen deaktiviert sind.
Datenschutz-Hinweis: Wenn der KI-Ernährungscoach-Schalter ausgeschaltet ist, wird Ihr Anbieter nur kontaktiert, wenn Sie aktiv ein Lebensmittelfoto aufnehmen. Es werden keine Daten für das Coaching oder die täglichen Zusammenfassungen gesendet — diese werden durch integrierte Vorlagen erstellt, die vollständig auf Ihrem Gerät ausgeführt werden.

Häufig gestellte Fragen (FAQ)

Ist mein API-Schlüssel sicher?

Ja. Ihr API-Schlüssel wird sicher im nativen sicheren Speicher Ihres Geräts (iOS Keychain / Android Keystore über capacitor-secure-storage-plugin) gespeichert und niemals im standardmäßigen lokalen Speicher oder in IndexedDB abgelegt. FuelFit hat keinen Backend-Server — es ist eine reine Client-Side-App. Alle KI-Aufrufe gehen direkt von Ihrem Gerät an den von Ihnen gewählten Anbieter. Wir erfassen oder sehen Ihre Daten niemals.

Was ist bei einem Gerätewechsel?

FuelFit enthält eine integrierte Export/Import-Funktion in den Einstellungen. Exportieren Sie Ihre Daten in eine Datei und importieren Sie sie auf Ihrem neuen Gerät. Ihre Mahlzeiten, Einstellungen und (verschlüsselten) API-Schlüssel werden in einem Schritt übertragen.

Kann ich KI nur für Fotos verwenden, nicht für das Coaching?

Ja. Öffnen Sie Einstellungen → KI-Dienste und schalten Sie den KI-Ernährungscoach aus. Dies deaktiviert die KI-generierten Coaching-Nachrichten, während die KI-Fotoanalyse aktiv bleibt.

Kann ich mehrere Anbieter nutzen?

FuelFit verwendet jeweils einen Hauptanbieter, aber Sie können mehrere benutzerdefinierte Anbieter konfigurieren und in den Einstellungen wechseln. Die drei integrierten Anbieter (Qwen, OpenAI, Mistral) merken sich ebenfalls ihre eigene Konfiguration.

Was passiert, wenn mein Guthaben aufgebraucht ist?

Die App zeigt eine Fehlermeldung an, wenn Sie versuchen, ein Foto zu analysieren oder wenn der Coach eine Nachricht generieren möchte. Wenn der Coach-Aufruf fehlschlägt, wechselt die Insights-Karte lautlos zu einem Vorlagen-Tipp (kein Fehler-Toast — und das "KI"-Badge verschwindet). Laden Sie Ihr Guthaben auf und der nächste Aufruf funktioniert wieder. Ihr Mahlzeitenverlauf wird lokal gespeichert.

Benötige ich einen API-Schlüssel, um FuelFit zu nutzen?

Nein. FuelFit funktioniert ohne KI für die manuelle Protokollierung. Sie benötigen nur dann einen API-Schlüssel, wenn Sie die Fotoanalyse oder KI-generiertes Coaching wünschen.

Welchen Anbieter soll ich wählen?

  • Standard: Qwen qwen3-vl-flash — am günstigsten, starke Essenserkennung
  • Am zuverlässigsten: OpenAI gpt-4o-mini — berechenbares Verhalten, breiter Support
  • EU-basiert: Mistral mistral-small-latest
  • Sonstige (Claude, Gemini, selbstgehostet): Custom-Option, am besten über OpenRouter

Wie wechsle ich den Anbieter?

Gehen Sie zu Einstellungen → KI-Dienste → Tippen Sie auf einen anderen Anbieter → Geben Sie den API-Schlüssel ein → Tippen Sie auf Verbindung überprüfen.

Warum ist das Modell-Feld für integrierte Anbieter editierbar?

Die Standardeinstellungen sind auf Kostenoptimierung ausgelegt. Wenn Sie eine höhere Genauigkeit wünschen, bevorzugen Sie möglicherweise qwen3-vl-plus bei Qwen oder gpt-4o bei OpenAI. Die Felder sind editierbar.
